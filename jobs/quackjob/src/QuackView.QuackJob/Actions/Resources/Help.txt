QuackJob
Actions:
  - run: Executes the specified job.
  - list: Lists all available jobs.
  - help: Displays this help information.

Usage:
  quackjob <action> [options]

Examples:
  quackjob run upcoming-calendar-events:outlook-calendar-import.json


Future concepts:
  quackjob run upcoming-calendar-events --account=... --calendar=... --days-in-future=14 --append
